# Jogo Gourmet

O Jogo Gourmet é um aplicativo desktop baseado na lógica de um jogo de adivinhação, similar ao Akinator, onde o programa busca descobrir o prato que o usuário está pensando através de uma série de perguntas.
O programa foi escrito usando **.NET 8.0** (com **Windows Forms**) e linguagem **C#**. 

## Funcionamento

O funcionamento do Jogo Gourmet é bastante simples e intuitivo para o usuário. Quando o jogo é iniciado, o programa faz perguntas ao jogador sobre características do prato que ele está pensando. Com base nas respostas fornecidas pelo usuário, o programa tenta adivinhar o prato em questão.
